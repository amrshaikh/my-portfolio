# Amr Shaikh - Portfolio

My personal portfolio website, designed and developed from scratch to showcase my work in web development and graphic design.

[**View Live Website**](https://amrshaikh.me)

---

## 🚀 Overview

This project is a single-page application built with vanilla web technologies.  
It features a clean, responsive design with a focus on performance and user experience.

### Key Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices  
- **Performance**: Optimized for fast load times using lazy loading and vanilla JavaScript  
- **Interactive UI**: Custom JavaScript image carousel, modal lightboxes, and smooth scroll navigation   

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript   
- **Styling**: Custom CSS with modern Flexbox/Grid layouts
- **Icons**: Google Material Symbols, Font Awesome, and Simple Icons (SVG) 
- **Fonts**: Inter (Google Fonts)
